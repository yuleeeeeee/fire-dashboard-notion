# PRD: FIRE 달성 계산기 & 진행률 대시보드

**작성일:** 2026-05-08  
**버전:** v1.0

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | FIRE 달성 계산기 (FIRE Progress Dashboard) |
| 목적 | Notion을 CMS로 활용한 자산 관리 및 FIRE 진행률 시각화 |
| CMS 선택 이유 | Notion API를 통해 비개발자도 데이터 입력·수정 가능, 별도 어드민 UI 불필요 |
| 목표 사용자 | FIRE(Financial Independence, Retire Early)를 목표로 자산을 관리하는 개인 |

---

## 2. 주요 기능

### 2.1 자산 입력 & 관리
- Notion DB에서 자산 데이터를 읽어 웹에 표시
- 자산 유형별(현금, 주식, 부동산 등) 분류 및 합산

### 2.2 FIRE 달성률 계산 & 시각화
- `달성률 = 총자산 / 목표금액 × 100`
- `FIRE 배수 = 총자산 / (월간생활비 × 12)`
- 목표까지 남은 금액 및 예상 달성 기간 표시

### 2.3 월별 추이 그래프
- 월별 총자산 변화 라인 차트
- 저축액 vs 투자수익 비교 바 차트

## 4. Notion 데이터베이스 구조

### 자산 목록 DB (`assets`)
| 필드명 | 타입 | 설명 |
|--------|------|------|
| 자산명 | Title | 자산 이름 |
| 금액 | Number | 현재 평가금액 (원) |
| 자산유형 | Select | 현금 / 주식 / ETF / 부동산 / 기타 |
| 수익률 | Number | 연환산 수익률 (%) |

### 월별 기록 DB (`monthly_records`)
| 필드명 | 타입 | 설명 |
|--------|------|------|
| 월 | Date | 기록 기준 월 (YYYY-MM-01) |
| 총자산 | Number | 해당 월 말 총자산 (원) |
| 저축액 | Number | 해당 월 저축 금액 (원) |
| 투자수익 | Number | 해당 월 투자손익 (원) |

### 목표 설정 DB (`goals`)
| 필드명 | 타입 | 설명 |
|--------|------|------|
| 목표금액 | Number | FIRE 달성 목표 자산 (원) |
| 월간생활비 | Number | 은퇴 후 월 생활비 (원) |

## 5. 화면 구성

### 5.1 메인 대시보드 (`/`)
- FIRE 달성률 프로그레스바 (대형)
- 핵심 지표 카드: 총자산 / 목표금액 / 달성률 / FIRE 배수
- 월별 총자산 추이 라인 차트

### 5.2 자산 목록 페이지 (`/assets`)
- 자산 테이블: 자산명 / 금액 / 유형 / 수익률
- 자산 유형별 합계 표시

### 5.3 분석 페이지 (`/analysis`)
- 자산 구성 비율 파이 차트
- 저축액 vs 투자수익 월별 비교 바 차트
